## Sintaxe básica
```cpp
int main() {}
```
- Aqui é definida uma função chamada `main`, sem parâmetros e que não retorna nenhum valor.
  
- As `{}` indicam agrupamento em C++, determinando o início e fim de cada função.

- Cada programa em C++ deve ter exatamente uma função global chamada `main`. O programa inicia executando essa função. O valor inteiro `ìnt` retornado pela função `main`, se houver, é o valor de retorno "para o _sitema_". Caso a `main` retorne qualquer valor diferente de zero, isso indica erro.

[EXIT_SUCCESS, EXIT_FAILURE]("https://en.cppreference.com/w/cpp/utility/program/EXIT_status.html")

## Hello World em C++ :
```cpp
#include <iostream>
     
int main() {
    std::cout << "Hello, World!\n";
}
```

  - `#include` $\rightarrow$ Declaração de bibliotecas usadas no programa.
  - `<iostream>` $\rightarrow$ Essa biblioteca fornece *objetos* que podem ler dados de entrada e saída do usuário para o console ou para um arquivo.
  - `< >` $\rightarrow$ Significa que ele está localizdo no sistema em algum lugar.

## Tipos de dados
C++ é uma linguagem fortemente tipada. Ou seja, cada variável no programa possui um tipo associado a ela, que define o que ela pode armazenar. O compilador realiza verificação rígida dos tipos ao compilar um programa em C++ para assegurar que o tipo correto esteja sendo usado e que os tipos sejam compatíveis quando combinados entre si (por exemplo, em cálculos).
